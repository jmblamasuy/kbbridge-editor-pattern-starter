# reference/ (local only — git-ignored)

Place here the **.NET sources of the GeneXus pattern** you are transcribing (for example,
the contents of `<GeneXus Platform SDK>\Patterns\<YourPattern>\Source` and `\Definitions`).

These files are **git-ignored** and must **never** be committed or published — only your
original TypeScript transcription ships. The folder exists so an AI assistant (or you) can
read the C# side-by-side while writing the TypeScript, annotating each class with a
`// Equivalent to <C# file> lines N-M` comment for traceability.

See `ai/CSHARP-TO-TYPESCRIPT-WORKFLOW.md` for the method.
